# Unsupported Or Uncertain Features

Verify current Apple container docs and local CLI help before using these Docker-derived features:

- Docker Compose full parity.
- Docker networks.
- Docker volumes.
- Docker buildx.
- Docker swarm.
- Docker plugins.
- Docker Desktop-specific features.
- Daemon socket compatibility.
- Linux-only Docker assumptions.
- Platform and architecture flags.

Do not say these are unsupported forever. Say that support must be verified before use. If unsupported, document the limitation and suggest a safe fallback or manual workaround.
